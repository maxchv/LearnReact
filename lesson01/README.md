# Install react

Start new app using `create-react-app` package:

```cmd
$ npm install create-react-app -g
$ create-react-app react-app
```

Run app:

```cmd
$ npm start
```

Start new .Net Core app with `react.js`:

```cmd
$ mkdir core-react-app
$ cd core-react-app
$ dotnet new react
```
